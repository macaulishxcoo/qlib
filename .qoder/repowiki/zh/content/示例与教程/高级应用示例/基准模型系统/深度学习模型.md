## LSTM 模型分析

LSTM（长短期记忆网络）是处理时间序列数据的经典RNN变体，具有门控机制来解决梯度消失问题。

```mermaid
classDiagram
class LSTMModel {
+input_size : int
+hidden_size : int
+num_layers : int
+dropout : float
+batch_first : bool
+forward(x) Tensor
+init_hidden(batch_size) Tensor
}
class LSTMLayer {
+input_gate : Gate
+forget_gate : Gate
+output_gate : Gate
+cell_gate : Gate
+forward(x, h_prev, c_prev) tuple
}
class Gate {
+weight_ih : Tensor
+weight_hh : Tensor
+bias : Tensor
+sigmoid() Tensor
+tanh() Tensor
}
LSTMModel --> LSTMLayer : "包含多个层"
LSTMLayer --> Gate : "使用门控机制"
```

GRU（门控循环单元）是LSTM的简化版本，通过更新门和重置门来控制信息流动。

